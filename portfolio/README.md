# Portfolio Project

This is a portfolio project that showcases various works and skills. Below are the details regarding the structure and usage of the project.

## Project Structure

```
portfolio
├── src
│   ├── index.html        # Main HTML document for the portfolio
│   ├── styles
│   │   └── input.css     # Tailwind CSS directives
│   └── main.js           # Main JavaScript file for interactivity
├── postcss.config.mjs    # Configuration for PostCSS with Tailwind CSS
├── tailwind.config.mjs    # Tailwind CSS configuration file
├── package.json           # npm configuration file
├── .gitignore             # Files and directories to be ignored by Git
└── README.md              # Documentation for the project
```

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Build the project**:
   ```bash
   npm run build
   ```

4. **Start the development server**:
   ```bash
   npm start
   ```

## Usage

- Open `src/index.html` in your browser to view the portfolio.
- Modify `src/styles/input.css` to customize styles using Tailwind CSS.
- Update `src/main.js` to add interactivity and functionality.
